# 什么是多线程
首先要了解线程和进程的概念。
## 线程
线程是操作系统能够进行运算调度的最小单位。被包含在进程之中，是进程中实际运作单位。
## 进程
程序的基本执行实体。一个软件运行之后就是一个进程。 


## 并发
在同一时刻，有多个指令在单个 CPU 上交替执行
## 并行
在同一时刻，有多个指令在多个 CPU 上同时执行

## CPU 核心
2核4线程
